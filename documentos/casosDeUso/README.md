
# Actores y Casos de Uso

## Introducción

Este documento describe los actores del sistema y los casos de uso asociados, definiendo las interacciones básicas entre los actores coordinador e investigador y el sistema mediante casos de uso atómicos alineados con el patrón CRUD.

## Propósito

Identificar actores

Definir casos de uso

Establecer responsabilidades
